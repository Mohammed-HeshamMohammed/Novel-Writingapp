# 📝 Novel-Writingapp – Your Offline AI-Powered Writing Assistant

**Novel-Writingapp** is an offline, privacy-respecting, AI-powered assistant for novelists, storytellers, and roleplay writers. Built with a React + Tailwind frontend and a FastAPI backend, it’s designed to help you organize chapters, get writing suggestions on demand, and build rich, immersive stories — all under your control.

---

## ⚙️ Features

- 🧠 **AI Suggestions (On-Demand)**  
  Use AI only when you choose. Get help with plots, character dialogue, grammar polish, or roleplay replies.

- 🗂 **Chapter & Scene Manager**  
  Structure your stories into scenes, arcs, and full chapters using a clean interface and local file system.

- 🎭 **Roleplay & Script Mode**  
  Switch between prose and roleplay formatting styles. Useful for game writers, RP groups, or collaborative story building.

- 📥 **Offline & Private**  
  All processing is local. No cloud, no tracking. You own your work completely.

- 🔁 **Import/Export Tools**  
  Easily bring in old stories or export your novel as `.txt`, `.md`, or `.pdf` with full formatting.

- 🧾 **Smart Memory System**  
  Save character bios, lore, and past AI interactions so suggestions are always context-aware.
